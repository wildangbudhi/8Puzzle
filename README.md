# 8Puzzle - Solver

AI - Implemented DFS, DLS, BFS, Dijsktra, and A* to solve the 8-puzzle problem.

## Algorithms Option :
- DFS Algorithm (Depth First Search)
- DLS Algorithm (Depth Limited Search)
- BFS Algorithm (Breadth First Search)
- Dijsktra Algorithm
- A* Algorithm

## Languange Used :
```
Python 3.7.1
```
